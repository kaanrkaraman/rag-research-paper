# Pinned dependencies for the BGE-M3 H100 reproducibility run.
#
# Usage on a fresh Lambda Labs / Vast.ai / RunPod H100 / H200 instance:
#     pip install -r requirements-h100.txt
#     pip install -e .            # exposes src/ to sys.path
#
# Notes:
# - PyTorch is installed from the cu124 wheel index. If the rented host
#   ships CUDA 12.1 instead of 12.4, swap the index URL to cu121.
# - faiss-cpu is intentional: FAISS search over 7,318 docs is CPU-fast and
#   faiss-gpu has flaky CUDA 12 wheels at the time of writing.
# - sentence-transformers 3.x is required for the CrossEncoder.predict
#   batch_size kwarg. 5.x also works but introduces minor API churn.
